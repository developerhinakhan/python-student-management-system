# Python Student Management System

A simple Python **OOP-based** project for managing and searching student records using a roll number.

## Features

* Uses Python Classes (OOP)
* Stores student information
* Search student by roll number
* Displays student details

## Tech Stack

* Python 3

## Run the Project

```bash
git clone https://github.com/developerhinakhan/python-student-management-system.git
cd python-student-management-system
python student_system.py
```
